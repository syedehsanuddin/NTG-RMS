# RMS Landing Page

A modern landing page for RMS (Restaurant Management System) by NTG Clarity Networks Inc.

## Pages

- **Home** (`index.html`) - Main landing page with features, testimonials, and CTA
- **Pricing** (`pricing.html`) - Pricing plans and comparison table
- **Help** (`help.html`) - Support, documentation, and help resources

## Setup

1. Place your logo file (`logo.png`) in the root directory
   - The logo should be the NTG Clarity Networks Inc. logo
   - Recommended size: height of 50px (width will scale automatically)

2. Open `index.html` in your web browser to view the site

## Features

- Modern, clean design inspired by Odoo
- White and cream color scheme
- Fully responsive (mobile, tablet, desktop)
- Smooth animations and transitions
- Mobile-friendly navigation menu

## Customization

### Colors
The color scheme can be customized in `styles.css` by modifying the CSS variables:
- `--primary-color`: Main brand color (currently purple)
- `--cream`: Cream background color
- `--text-dark`: Dark text color

### Content
All content can be edited directly in the HTML files. The pages are structured with semantic HTML for easy customization.

## Browser Support

- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)

## Notes

- The logo file (`logo.png`) needs to be added to the root directory
- All links to external resources (GitHub, etc.) are placeholder links and should be updated
